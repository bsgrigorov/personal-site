How to create your OG image:

  1. Open Chrome DevTools (F12 or Cmd+Option+I)
  2. Click the device toolbar icon or press Cmd+Shift+M
  3. Set dimensions to 1200 x 630 pixels
  4. Right-click on the OG image above and select "Capture node screenshot"
  5. Save as og-image.png in public/images/
  6. Optionally compress the image using TinyPNG or similar
